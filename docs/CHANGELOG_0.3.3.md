# v0.3.3

- Fixed reverse SyncTeX: double-clicking a PDF now moves the TeX cursor without clearing/reloading the current PDF preview.
- Added PDF preview render-quality setting in Settings. Lower values improve performance; higher values improve clarity.
- PDF preview now reacts to Ctrl/⌘/Alt + wheel, deltaZ touchpad pinch events, and WebKit gesture events when available.
- Image preview now supports the same trackpad gesture path.
