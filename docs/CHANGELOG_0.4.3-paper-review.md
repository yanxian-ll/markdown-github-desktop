# v0.4.3 Paper Review

本版继续打磨 Overleaf 式 PDF/源码交互体验，并重点修复浅色模式和 PDF 预览空间占用问题。

## PDF 预览体验

- PDF 默认启用“适应宽度”模式，打开和重新编译后会自动按预览栏宽度居中显示。
- 保留手动缩放逻辑：点击 `-` / `+` / `1:1` 或触控板/鼠标滚轮缩放后，会自动切换到手动模式。
- 新增 `适宽` 按钮，可随时恢复自动适应宽度。
- 编译或刷新 PDF 后尽量保留当前页码，避免每次回到第一页。
- 压缩 PDF 页与页之间的间距，移除每页外层的大块留白。
- 页码标签改为轻量浮层，避免占用页面上方空间。

## SyncTeX 定位交互

- PDF 定位点增加短暂脉冲动画，源码跳 PDF 时更容易看到目标位置。
- PDF 双击反向定位到 TeX 后，源码目标行会短暂闪烁高亮。
- 连续定位到同一源码行也会重新触发滚动和高亮。

## 浅色模式

- 提高浅色模式下文字、按钮、工具栏、侧栏、批注面板、CodeMirror gutter 的对比度。
- PDF 工具栏、批注卡片、状态文字在浅色模式下更清楚。

## 兼容性

- 保留 v0.4.2 的批注 JSONL 存储方式：`.paper-notes/annotations.jsonl`。
- 保留 PDF 区域批注、源码行批注、GitHub 提交方式。
