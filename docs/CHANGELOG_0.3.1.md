# v0.3.1

## 改进

- PDF 预览默认连续滚动，普通滚轮浏览，Ctrl/⌘ + 滚轮缩放。
- PDF 渲染改为低清晰度、可见区域按需渲染，避免大 PDF 打开和翻页卡顿。
- 图片预览支持 Ctrl/⌘ + 滚轮缩放。
- LaTeX 构建在 Tauri 后端阻塞线程池中执行，不占用前端交互线程。
- 为 clone / pull、刷新状态、提交、LaTeX 构建增加防重复点击锁。

## 注意

- 低清晰度 PDF 预览用于提高编辑流畅度；最终导出的 PDF 文件不会受影响。
- 如果需要更高预览清晰度，可后续将 `PdfPreview.vue` 中 `LOW_RES_FACTOR` 调高。
