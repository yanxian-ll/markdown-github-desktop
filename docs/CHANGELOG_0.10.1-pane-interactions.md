# v0.10.1 Pane interactions and resizable logs

- 保持研究流侧栏状态：切换文件时，如果左侧编辑辅助栏处于“研”模式，不再自动关闭；大纲、文献、片段、历史等文件相关面板仍会在切换文件时收起。
- 将编辑栏与预览栏改成更对等的 pane 模式：Markdown / LaTeX 文件默认同时打开编辑与预览。
- 支持双击栏顶部切换布局：
  - 双栏同时打开时，双击编辑栏顶部会关闭编辑栏，保留预览；双击预览栏顶部会关闭预览，保留编辑。
  - 只显示编辑栏时，双击编辑栏顶部会打开预览栏；只显示预览栏时，双击预览栏顶部会打开编辑栏。
- 新增 pane 快捷键：
  - `Ctrl/Cmd + Alt + V`：切换预览栏。
  - `Ctrl/Cmd + Alt + E`：切换编辑栏。
  - `Ctrl/Cmd + Alt + \`：恢复编辑 + 预览双栏。
- 移除顶部工具栏中的全局预览开关，改为在编辑栏 / 预览栏内部操作。
- 日志 / 问题 / 输出底部面板新增顶部拖拽手柄，可向上或向下调整高度。
