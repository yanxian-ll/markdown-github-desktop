# v0.4.5 Paper Review

本版本把 PDF 批注交互从“只能拖框”扩展为“文字批注优先、区域批注保留”。

## 新增

- PDF 默认模式下支持选中文字后浮动工具条批注。
- 选中 PDF 文字后可直接执行：批注、高亮、定位源码。
- 选中 PDF 文字后右键可呼出同一批注操作。
- 文字批注保存 `textQuote` 和多行归一化文字矩形，仍写入 `.paper-notes/annotations.jsonl`。
- 原顶部“批注”按钮改名为“区域批注”，用于图、表、公式和版式问题。
- 论文批注栏支持拖拽调整宽度。
- 论文批注栏支持隐藏和重新显示，隐藏后 PDF 自动占满空间。

## 保留

- PDF 双击反向定位 TeX。
- 区域框选批注。
- 源码行批注。
- 批注状态、时间筛选、Git sidecar 保存。
- PDF 自动适宽、居中、手动缩放。

## 验证

- `npx vue-tsc --noEmit` 通过。
- `npx vite build` 通过。
