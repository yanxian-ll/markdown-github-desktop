# Changelog 0.2.4

## 文件树交互

- 左侧恢复为单一“＋ 新建”入口，不再分别放置“新建 MD / 新建 TeX / 新建文件夹”多个按钮。
- “＋ 新建”支持输入常用相对路径：`.md`、`.tex`、`.txt`、`.bib`、`.sty`、`.cls` 等。
- 输入以 `/` 或 `\` 结尾的路径会创建文件夹，例如 `notes/`。
- 左侧目录树支持文件夹展开、文件打开、文件夹内新建、重命名、删除。
- Git clone / pull 完成后会自动刷新左侧目录树，直接显示本地仓库结构。

## Git 行为

- 仍保持本地 Git 工作区模型：Ctrl/Cmd+S 只保存本地；手动点击“提交 GitHub”才执行 `git add -A && git commit && git push`。
- 删除、重命名、新建文件都会通过 Git 状态显示并参与手动提交。
