# v0.4.1-paper-review

本版本在 v0.4.0 的论文批注基础上继续调整交互：

- PDF 预览默认进入统一的“定位/文字模式”：可选择文本，也可双击 PDF 反向定位到 TeX。
- 不再把“文本”和“定位”做成两个模式；顶部只保留“批注/退出批注”开关。
- PDF 控件固定在 PDF 区域最上方，滚动论文时仍能操作页码、缩放和批注开关。
- 点击“批注”后进入框选批注状态；退出批注后恢复默认定位/文字状态。
- 调整预览区拖拽宽度限制，向左拖动时不再被 920px 上限卡住，只保留编辑区最小可用宽度。
- 批注侧栏新增时间筛选：今天、近 7 天、近 30 天、全部时间。
- 批注列表按日期分组，并显示创建/更新时间，方便批注记录很多时浏览。
- 界面明确说明批注保存到 `.paper-notes/annotations.jsonl`：提交 GitHub 后，其他设备用本软件打开同一项目可见；GitHub 网页直接预览 PDF 不会显示这些 sidecar 批注。

验证：

```bash
npm run build
```

当前环境没有 `cargo`，未执行 `cargo check`。本机安装 Rust 后建议运行：

```bash
cd src-tauri
cargo check
```
